# 📊 Attendance Portal Project

A modern web-based **Attendance Management System** designed to simplify student attendance tracking with a clean UI and efficient backend.

---

## 🚀 Live Demo  
🔗 https://attendance-portal-liard.vercel.app

---

## 📌 Features

- ✅ User Authentication (Login / Auto-login)
- 📅 Daily Attendance Tracking
- 🧑‍🎓 Student Management System
- 📊 Real-time Attendance Records
- ⚡ Fast & Responsive UI
- 🔐 Secure Backend Handling
- 🗂️ Organized Database Structure
