## 📌 About Project  

The **Attendance Portal** is a web-based system designed to manage and track student attendance efficiently.  
It provides a simple, fast, and user-friendly interface for handling attendance records in real time.

---

## 📌 Features

✨ Core Features:
- 🔐 User Authentication (Login / Auto-login)
- 📅 Daily Attendance Tracking
- 🧑‍🎓 Student Management System
- 📊 Real-time Attendance Records
- ⚡ Fast & Responsive UI
- 🔐 Secure Backend System
- 🗂️ Structured Database

✨ Developer Friendly:
- 📁 Clean Code Structure  
- ⚙️ Easy Setup  
- 🚀 Scalable System
